Wintel
=======

Wintel is a student project of phone book developped in first year of study of computer
science. It uses Swing, serializable classes and file I/O.

Usage
-------

    $ ant run

Compiles the sources and run the application

    $ ant testJUnit

Execute the test suite using JUnit, the library is already set in the CLASSPATH

    $ ant help

Display some more information about the project
